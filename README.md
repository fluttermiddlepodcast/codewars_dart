## How to run

First of all, make sure that you have correct Dart version installed on your system. [Link](./pubspec.yaml) contains information about minimum Dart version in `platform` section.

1. Install dependencies:

```shell
$ dart pub get
```

2. Run tests:

```shell
$ dart test -c source
```
